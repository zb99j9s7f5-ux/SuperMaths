# 🧮 SuperMaths

**A free, AI-powered mathematics tutor for students in Grades 1–12.**

SuperMaths adapts to each student's level, generates unlimited practice questions, and delivers instant step-by-step hints — all powered by AI. No subscriptions, no ads, no limits.

🔗 **Live app:** [supermaths.streamlit.app](https://supermaths.streamlit.app/)

---

## ✨ Features

- **Placement Test** — A 10-question diagnostic test that automatically determines the right starting level for each student
- **AI-Generated Questions** — Every question is freshly generated by Groq's LLaMA 3.3 70B model — no question bank, no repeats, infinite practice
- **Batch Generation** — All questions for a session are generated in a single API call, reducing load times to ~3–6 seconds
- **Level Assignments** — 5 topics per level, 2 quizzes each (5 questions per quiz), with progress tracked per topic
- **Level Test** — Score 70% or higher to skip straight to the next level without completing all assignments
- **Free Practice** — Unlimited topic practice at the current level, tracked toward daily totals
- **Instant Hints** — On-demand AI explanations that guide students through the method without giving away the answer
- **Badges** — 31 unlockable badges across five categories: Progress, Levels, Skill, Consistency, and Topic Mastery
- **Video Lessons** — Curated YouTube lessons mapped to every topic at every level
- **Daily Tracking** — Questions answered and accuracy tracked per day with a visual progress bar
- **Streak System** — Consecutive daily activity tracked and rewarded with streak badges
- **Persistent Progress** — All progress, badges, and scores saved to MongoDB Atlas

## 🚀 Running Locally

**1. Clone the repository**
```bash
git clone https://github.com/your-repo/supermaths.git
cd supermaths
```

**2. Install dependencies**
```bash
pip install -r requirements.txt
```

**3. Configure environment variables**

Create an `api.env` file in the project root:
```env
GROQ_API_KEY=your_groq_api_key
MONGO_URI=your_mongodb_connection_string
GROQ_MODEL=llama-3.3-70b-versatile
```

**4. Run the app**
```bash
streamlit run main.py
```
